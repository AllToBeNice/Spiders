# Spiders
随随便便写的一些网络爬虫

## 新型冠状病毒疫情分布图
`新型冠状病毒疫情分布图.py`：主程序，
`./data/Pneumonia.db`：数据库文件，
`pneumoniaPic.png`：数据来源方绘制的实时图片（需要通过网页获取图片URL，再通过程序获取），
`TimeRun.ps1`：使用Power Shell持续运行主程序，在分钟数为`00`或`30`时执行主程序获取数据。

## 网易严选商品评论小爬虫
`spider.py`：主程序，
`DataAna.py`：统用数据分析程序，
`speAna.py`：特例数据分析程序，
`./data/*.db`：数据文件。
